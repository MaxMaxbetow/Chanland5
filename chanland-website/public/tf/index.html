🏛️ О федерации

Торговый центр сервера

Этот сайт можно использовать как официальный прайс-лист, витрину и точку входа для игроков. Дизайн сделан в той же атмосфере, что и твой основной сайт: стекло, мягкий свет, округлые карточки и большой акцент на визуал.

Поиск Находи предметы по названию — хоть элитры, хоть Починку, хоть Тишину, хоть зловещие ключи.
Категории Оружие, броня, декор, механизмы, еда, строительство, транспорт и прочие полезности.
Витрина Каждый товар в отдельной карточке, а не просто как список из бездны текста. Глазам легче, мозгу приятнее.
Основа Один HTML-файл с CSS и JS внутри. Открыл — и сайт уже живой. Без React и без лишней возни.

Как это можно использовать

Этот блок я уже заложил под живую инфу. Здесь можно разместить правила покупки, способы оплаты, ник продавца, ссылки на Discord, время работы, наличие скидок и любые важные примечания.

🪙

Валюта

Сейчас цены выводятся как обычные числа. При желании можно добавить рядом название валюты сервера или иконку монет.

📩

Оформление

Можно сделать кнопку «Заказать» на каждой карточке и привязать её к Discord, форме или любому другому способу связи.
